### **git 常用命令**
---

#### ***1. 查看及设置用户签名*** 
> git config --global --list  
> git config --global user.name 用户名  
> git config --global user.email 邮箱
---

#### ***2. 初始化本地库*** 
> git init
---

#### ***3. 查看本地库状态*** 
> git status
---

#### ***4. 暂存区添加及取消*** 
> git add 文件名  
> git rm --cached 文件名
---

#### ***5. 提交本地库*** 
> git commit -m "commit content" 文件名
---

#### ***6. 查看历史版本*** 
> git reflog  
> git log
---

#### ***7. 版本穿梭*** 
> git reset --hard 版本号
---
