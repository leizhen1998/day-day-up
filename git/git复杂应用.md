### **一、git 常用命令**
---

#### ***1. 查看修改之后还未暂存的变化内容*** 
> git diff
---

#### ***2. 对比已暂存文件与最后一次提交的文件差异*** 
> git diff --staged  
> git diff --cached
---

#### ***3. 提交所有已追踪过文件（省略暂存操作）*** 
> git commit -a -m "commit content"
---

#### ***4. 仅将文件移除暂存区，保留在当前工作目录*** 
> git rm --cached 文件名
---

#### ***5. 将文件从暂存区和当前工作目录移除*** 
> git rm 文件名
---

#### ***6. 重新提交*** 
> git commit --amend
---
